DESCRIZIONE DEI PAYLOAD:

VTXHEN = consente l'esecuzione di contenuti firmati falsi, aggiunge il debug settings.

MIRAHEN = consente l'esecuzione di contenuti firmati falsi, aggiunge impostazioni di debug, avvia UART per MIRA sulla porta 9999/9998, esegue due volte la funzione Bin Loader.

MIRA = abilita l'accesso al kernel (UART) tramite telnet

BINLOADER = consente di inviare un payload.bin sulla tua PS4.

MIRA+BIN = esegui una volta Mira, due volte per Bin Loader..

BLOCKER + FTP = blocca gli aggiornamenti indesiderati (perma) e viene avviato un server ftp temporaneo.

BLOCK EX = blocca tutte le possibili posizioni dell' aggiornamento.

UNBLOCKER = consente di nuovo gli aggiornamenti.

UNBLOCK EX = annulla quale blocco deve essere inserito.

EEPLINUX = avvia un kernel Linux leggero dalla tua chiavetta USB

LINUX = avvia un kernel Linux dalla tua chiavetta USB.

KEXEC = consente di avviare nuovamente Linux da PS4 OS (orbis)

VTXDUMP = consente di dumpare i propri giochi legittimi (disco / psn) in USB hdd, con o senza patch e uniti / non raggruppati.

DB_SG = esegue il backup di tutti i database critici, salva i giochi e i dati dei trofei su USB..

APPTOUSB = sposta i giochi su qualsiasi USB esterno (v3 puoi specificare il punto di montaggio), aggirando l'unità interna che sta esaurendo lo spazio.

APPTOUSB_ALT = sposta i giochi su USB1 esterno, aggirando l'unità interna esaurendo lo spazio..

USB_FPKG = alternativa basata su pkg2usb da SiSTRO e AppToUsb da stooged

KERNELDUMP = consente di effetuare il dump del kernel su USB (4.05-5.05)

KERNELDUMPER = consente di effetuare il dump del kernel su USB

FS_DUMPER = esegue il dump del filesystem PS4 su USB.

VR_ENABLER = abilita il VR se ti sei dimenticato di abilitarlo prima di eseguire HEN

VTXFTP = avvia un server ftp su PS4 in modo da poter collegare e copiare / incollare file sul / dal tuo PC/PS4.

FTP+R/W = come server ftp ma con privilegi R / W completi su tutte le partizioni PS4.

REACTPSPLUS = imposta il RTC al 2012 in modo da poter giocare ai giochi PSN leggittimi bloccati (UNREACTPSPLUS torna al 2018).

REACTPSPLUSMODDABLE = imposta l'RTC su qualsiasi data desiderata per poter giocare ai giochi PSPLUS scaduti.

PS4TRAINER = ascolta i file .cht da caricare per i tuoi giochi.

PS4CHEATER = ascolta i file .cht da caricare per i tuoi giochi.

PS4API = server API per modding in tempo reale (RTM)

APII_INTENSE_EU_127 = mod menu for only EU GTAV V1.27

LAMANCE_127 = mod menu per US/EU GTAV V1.27

LAMANCE = mod menu per US/EU GTAV V1.00

PSYC0S = mod menu per US/EU GTAV V1.00

APII_INTENSE = mod menu per US/EU GTAV V1.00

WILDEMODZ = mod menu per US/EU GTAV V1.00

TRAINER = mod menu per US/EU GTAV V1.00

NOTANOTHER = mod menu per US/EU GTAV V1.00

ARABICGUY = mod menu per US/EU GTAV V1.00

Black OPS 3 = payload LAN party (necessita di connessione internet)

CACHE_INSTALL= copia ApplicationCache.db dalla root USB alla cache del browser, crea una cache di base se USB non viene trovata.

USB_CACHE_INSTALL= copia ApplicationCache.db solo dalla root USB.

APP_CACHE = installa un ApplicationCache.db che è una versione modificata di PEH di Al-Azif sul tuo browser.

HISTORY_BLOCKER = attiva / disattiva il caricamento automatico dell'ultima pagina, mostra invece "pagine usate frequentemente".
